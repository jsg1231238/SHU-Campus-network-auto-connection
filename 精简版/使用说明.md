# 校园网自动认证 · 使用说明

这个脚本能让电脑自动帮你完成校园网登录，掉线了会自动重连，不用你手动打开网页输密码。**前提：你和给你这份脚本的人是同一所学校、连的是同一套认证系统**（如果不确定，问问对方）。

文件夹里有两个文件，缺一不可，请始终放在同一个文件夹里：
- `campus-net-api.ps1` —— 真正干活的脚本
- `launch-hidden.vbs` —— 让脚本在后台安静运行、不弹黑窗口的小工具

---

## 第一步：放到你电脑上

把整个文件夹拷贝到你电脑上任意位置，比如 `D:\campus-net\`（后面步骤会用到这个路径，记住它）。

## 第二步：改成你自己的信息

用**记事本**打开文件夹里的 `campus-net-api.ps1`（右键 → 打开方式 → 记事本），找到最上面 `param(` 后面的两行：

```powershell
[string]$WifiSSID   = "改成你的WiFi名字",
[string]$Account    = "改成你的校园网账号(学号)",
```

把两个双引号里的内容分别改成：
- 你连接的 WiFi 名字（在电脑右下角网络图标里能看到）
- 你的校园网账号（一般是学号）

**下面的 `$Service`、`$PortalHost` 以及那一长串字母数字的 `$PubMod` 都不用改**——这些是你们学校认证系统本身的设置，全校通用。改完保存、关闭记事本。

## 第三步：存一次你的密码

打开开始菜单，搜索 **PowerShell**，直接打开（不需要管理员权限）。

1. 输入 `cd ` 后**空一格**，把你的文件夹从资源管理器里拖进这个窗口（路径会自动填上），回车。比如：
   ```
   cd D:\campus-net
   ```
2. 复制下面这一整行，粘贴进去，回车：
   ```powershell
   (Read-Host '校园网密码' -AsSecureString | ConvertFrom-SecureString) | Set-Content .\.campuspwd -Encoding ASCII
   ```
3. 提示"校园网密码:"时输入你的密码——**屏幕上不会显示任何字符，这是正常的**，输完直接回车。

这样密码会被 Windows 自带的加密功能处理后存进 `.campuspwd` 文件，**只有你这台电脑、你这个账号能打开它**，别人拿走这个文件也没用。

## 第四步：测试一次

先确保自己现在是"未登录"校园网状态（可以先手动登录一次，再点认证页上的"注销"）。然后在刚才的 PowerShell 窗口里输入：

```powershell
powershell -ExecutionPolicy Bypass -File .\campus-net-api.ps1
Get-Content .\campus-net-log.txt -Tail 3
```

如果最后几行日志出现 `SUCCESS`，说明成功了，现在应该已经能上网。

如果看到"密码错误"或别的报错，**不要自己瞎改脚本**，把这几行日志截图，找给你这份脚本的人帮忙看。

## 第五步：设置成自动运行

开始菜单搜索 **PowerShell**，右键选**"以管理员身份运行"**（这一步必须管理员权限）。把下面整段复制粘贴进去，回车（把第一行路径改成你自己文件夹的实际路径）：

```powershell
$vbs = "D:\campus-net\launch-hidden.vbs"
$act = New-ScheduledTaskAction -Execute "wscript.exe" -Argument "//B //Nologo `"$vbs`""
$t1  = New-ScheduledTaskTrigger -AtLogOn
$t2  = New-ScheduledTaskTrigger -Once -At (Get-Date) -RepetitionInterval (New-TimeSpan -Minutes 30) -RepetitionDuration (New-TimeSpan -Days 3650)
$set = New-ScheduledTaskSettingsSet -AllowStartIfOnBatteries -DontStopIfGoingOnBatteries -StartWhenAvailable -ExecutionTimeLimit (New-TimeSpan -Minutes 3) -MultipleInstances IgnoreNew
Register-ScheduledTask -TaskName "CampusNetAutoLogin" -Action $act -Trigger $t1,$t2 -Settings $set -Description "校园网自动认证"
```

这里**只需要改第一行的路径**——`launch-hidden.vbs` 内部会自动找到同文件夹里的脚本，不用再单独改它。

做完之后，电脑会在**每次开机登录 + 之后每 30 分钟**自动检查一次网络，掉线了就自动重新登录，全程没有任何弹窗。

---

## 安全提醒

- `.campuspwd` 是你的加密密码文件，不要发给别人、不要截图。
- 如果以后要把这个文件夹再转发给别的同学，记得先删除你自己生成的 `.campuspwd` 和 `campus-net-log.txt` 这两个文件（都是运行后才会出现的，删了不影响别人重新配置）。
